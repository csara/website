My cookbook

August 5, 2014 by Chris

A short post about my cookbook

Cookbook

I love cooking. It's sort of necessary unless you want to eat out for every meal. So I collected recipes over the years. I got kind of tired of going through a list of files to find the recipe I wanted, so I deciced to make a cookbook! I love Latex, and I found an awesome Latex style called xcookybooky. With xcookybooky, you can create recipes in Latex and it formats the recipes in a fairly neat design.

The Details

I decided to throw the cookbook up on my GitHub. I figured people could add to this (sharing is caring!) or fork as a basis for their own cookbook if they want. Download or check out the files here. If you don't know Latex there's never a better time to learn than now! The .tex file is pretty well organized so it should be pretty simple to pick up. Anyway, enjoy!

Yeah first post! Weird. I never thought I would start a blog let alone host it myself.

Creating this site

I've thought about making a website for a while. I had no idea where to begin. Luckily with some research (Googling), and the help of my friend I was able to dive in to this project.

Bootstrap

Probably the best thing for web development ever. Well I guess if you're like me and want to stay a little more hands off design it is. Check out Bootstrap! It's seriously awesome. It's also Open Source and pretty industry standard!

Reasons for creating this site

  • To sharpen my web development skills.
  • Sharing my interests and knowledge.
  • Help you learn more about me!

The biggest thing I've learned is making a site requires a lot of writing! The "about me" section was an interesting section to write. It made me think about what my interests are and what I spend my time doing. It made me feel fortunate to live in a place where all these different hobbies are possible.
